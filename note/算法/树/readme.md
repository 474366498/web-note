## 二叉树 
1. 一个二叉树第I层的最大项目点数为2**(I-1) i >= 1
2. 深度为K的二叉树有最大节点总数为 2**K - 1 K >=1
3. 对任何非空二叉树T 若n0表示叶节点的个数、n2是度为2的非叶节点个数 那么两者满足关系 n0 = n2 + 1
