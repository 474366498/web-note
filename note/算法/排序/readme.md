### 插入排序
  1. 在数组中默认分成两部分 前面一部分(*_A_*)是有序排列好的(就是数组中下标为0的那项) 另一部分(*_B_*)是数组中的其它内容
  2. 在另一部分(*_B_*)中循环 
    a. 循环中的单项  如果比*_A_*大 就往*_A_*里添加 反之就插入到*_A_*之中
